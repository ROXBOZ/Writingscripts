Who is A11Y?

Web accessibility (or A11Y) allows people with disabilities - visual, auditory or cognitive - permanent or temporary - to navigate the web easily. The WCAG 2.1 AA standards(open a new tab) (Web Content Accessibility Guidelines) define a whole series of principles to be applied in order to ensure a perceptible, usable and understandable web for everyone.

Web accessibility is web inclusion. Code, design and content have to meets some standards for a website to be accessible. Web Accessibility increases the user experience â€“ everyone should benefit from it.

Text & Font CSS settings

Accessibility & Writing systems

Good practice in web typography and web accessibility means well adjusted CSS settings.

But, different writing systems mean different CSS adjustments. To mention just a few examples, paragraphs are commonly set at 16px in Latin, but the Arabic writing system needs 18px to reach satisfying readibility. Vietnamese is based on the latin writing system, but its accented glyphs can be much taller than those found in Western European languages; for this reason, it needs extra line-height to maintain readability. Cyrillic on the other hand adapts well to latin CSS settings.

Most websites hosted in Europe and the Americas translated in writing systems other than Latin are not set correctly. As a result, the user experience drops drastically, when switching the language.

Layout CSS settings

Language directions

Content are translated and, when the direction of the language changes, the layout must get inverted. However, most of the time, the layout keeps following the left-to-right logic in all languages (menu remains top right, logo remains top left, etc). An inclusive multilingual website should see its layout reverted to be correctly adapted to right-to-left writing scripts. It includes menus, tables, forms, etc.

Last but not least, not only the design but also the code has to be adaped; when navigating the website with a keyboard, the tab focus should follow the direction of the language.

This website aims to become a ressourceful tool when designing multilingual websites. For now, it's just a beginning.
